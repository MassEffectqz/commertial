Техническая шпаргалка

Frontend

React 19

Библиотека JavaScript для создания пользовательских интерфейсов. Версия 19 включает улучшения производительности и новые возможности.

TypeScript

Надмножество JavaScript с добавленной статической типизацией. Помогает находить ошибки на этапе разработки.

SSR (Server-Side Rendering)

Рендеринг React-компонентов на сервере. Улучшает SEO и скорость первой отрисовки страницы.

TailwindCSS

Утилитарный CSS-фреймворк. Позволяет стилизовать элементы прямо в HTML с помощью готовых классов.

Code Splitting

Разделение кода на несколько файлов (чанков). Ускоряет загрузку страницы.

Lazy Loading

Ленивая загрузка компонентов. Загружает код только когда он нужен пользователю.

Backend

Node.js

Среда выполнения JavaScript на стороне сервера. Позволяет использовать JS для бэкенда.

Express

Минималистичный веб-фреймворк для Node.js. Упрощает создание API и серверов.

RESTful API

Архитектурный стиль для API. Использует HTTP-методы (GET, POST, PUT, DELETE) для работы с ресурсами.

Middleware

Промежуточное ПО в Express. Обрабатывает запросы до попадания в основной обработчик.

База данных

PostgreSQL

Реляционная СУБД с открытым исходным кодом. Поддерживает сложные запросы и транзакции.

Миграции БД

Версионирование схемы базы данных. Позволяет применять изменения структуры поэтапно.

ORM

Object-Relational Mapping. Преобразует данные между объектами кода и таблицами БД.

Параметризованные запросы

SQL-запросы с параметрами. Защищают от SQL-инъекций.

Безопасность

JWT (JSON Web Token)

Стандарт для безопасной передачи данных между сторонами. Используется для аутентификации.

bcrypt

Функция хеширования паролей. Добавляет "соль" для защиты от радужных таблиц.

CSRF (Cross-Site Request Forgery)

Атака, заставляющая пользователя выполнить нежелательное действие. Защищается токенами.

Rate Limiting

Ограничение количества запросов от одного клиента. Защита от DDoS и брутфорса.

Helmet.js

Набор middleware для Express. Добавляет HTTP-заголовки безопасности (CSP, HSTS, X-Frame-Options).

CSP (Content Security Policy)

Политика безопасности контента. Ограничивает источники загружаемых ресурсов.

SQL Injection

Внедрение вредоносного SQL-кода через пользовательский ввод. Защищается параметризованными запросами.

XSS (Cross-Site Scripting)

Внедрение вредоносного JavaScript. Защищается экранированием вывода и CSP.

HTTPS

Безопасная версия HTTP с шифрованием TLS/SSL. Защищает данные при передаче.

HSTS

HTTP Strict Transport Security. Принудительно использует HTTPS для домена.

Производительность

Кэширование

Сохранение результатов вычислений для быстрого доступа. Уменьшает нагрузку на БД.

Gzip сжатие

Сжатие HTTP-ответов сервером. Уменьшает размер передаваемых данных.

Sharp

Библиотека для обработки изображений. Конвертирует в WebP, создаёт превью, ресайзит.

WebP

Современный формат изображений. На 25-35% меньше JPEG при том же качестве.

Debouncing

Задержка выполнения функции до прекращения ввода. Используется в поиске.

useMemo

Хук React для мемоизации значений. Кэширует результат вычислений.

useCallback

Хук React для мемоизации функций. Предотвращает пересоздание функций при ререндере.

SEO

Schema.org

Микроразметка для поисковиков. Добавляет структурированные данные о товарах, отзывах и т.д.

Sitemap.xml

Файл со списком всех страниц сайта. Помогает поисковикам индексировать контент.

Robots.txt

Файл с правилами для поисковых ботов. Указывает, какие страницы можно сканировать.

Open Graph

Метатеги для соцсетей. Контролируют, как ссылка отображается при шеринге.

Meta-теги

HTML-теги с информацией о странице (title, description, keywords). Важны для SEO.

Slug

Человекопонятный URL (например, /catalog/video-cards вместо /product?id=123).

Canonical URL

Указывает поисковикам на основную версию страницы при дублировании контента.

DevOps

Docker

Платформа контейнеризации. Упаковывает приложение со всеми зависимостями.

Docker Compose

Инструмент для запуска нескольких контейнеров. Упрощает развёртывание.

CI/CD

Continuous Integration / Continuous Deployment. Автоматизация сборки и развёртывания.

Nginx

Веб-сервер и обратный прокси. Раздаёт статику, балансирует нагрузку.

Environment переменные

Переменные окружения для хранения конфигурации (пароли, ключи API).

Let's Encrypt

Бесплатный центр сертификации SSL/TLS. Предоставляет сертификаты на 90 дней.

Тестирование

Vitest

Фреймворк для тестирования JavaScript. Быстрый, совместим с Jest.

Unit тесты

Тестирование отдельных функций/компонентов изолированно.

API тесты

Тестирование конечных точек API на корректность ответов.

UI тесты

Тестирование пользовательского интерфейса (React Testing Library).

Supertest

Библиотека для тестирования HTTP-запросов в Node.js.

Coverage report

Отчёт о покрытии кода тестами. Показывает, какой процент кода протестирован.

Работа с файлами

Multer

Middleware для загрузки файлов в Express. Обрабатывает multipart/form-data.

S3-совместимое хранилище

Облачное хранилище файлов по стандарту Amazon S3.

Множественная загрузка

Загрузка нескольких файлов одновременно (галерея товаров).

API

REST

Representational State Transfer. Архитектурный стиль для распределённых систем.

Версионирование API

Указание версии в URL (/api/v1/...). Позволяет поддерживать обратную совместимость.

Пагинация

Разделение данных на страницы (?page=1&limit=20). Уменьшает нагрузку.

HTTP коды статусов

200 OK, 201 Created, 400 Bad Request, 401 Unauthorized, 403 Forbidden, 404 Not Found, 500 Internal Server Error

Аутентификация

JWT Token

Токен доступа в формате JSON. Содержит payload с данными пользователя и подпись.

Хеширование паролей

Преобразование пароля в хеш. Необратимо, защищает при утечке БД.

Сессионные токены

Временные токены для аутентификации. Истекают через заданное время.

CORS

Cross-Origin Resource Sharing. Политика безопасности для запросов между доменами.

React хуки

useState

Добавляет состояние в функциональный компонент.

useEffect

Выполняет побочные эффекты (запросы к API, подписки).

useContext

Получает доступ к контексту React без пропсов.

useReducer

Альтернатива useState для сложного состояния с редюсером.

useRef

Создаёт изменяемую ссылку, сохраняющуюся между ререндерами.

useMemo

Мемоизирует значение для оптимизации производительности.

useCallback

Мемоизирует функцию для оптимизации производительности.

useNavigate

Хук React Router для программной навигации.
